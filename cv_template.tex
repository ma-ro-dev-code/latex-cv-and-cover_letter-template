% ---------------------------------------------------------
% DATE FORMAT CONVENTION
% ---------------------------------------------------------
% Dates in this template follow the format "mmm yyyy"
% (first three letters of the month + four-digit year).
%
% This format is intentionally chosen to:
% - keep all date ranges on a single line without wrapping,
% - ensure consistent visual alignment across entries,
% - remain language-neutral and internationally readable,
% - balance compactness with full clarity of the time period.
%
% Example: "Jan 2021 -- Dec 2023"
%
% Using full month names or numeric formats may cause
% line breaks or misalignment in the date column.
% ---------------------------------------------------------
