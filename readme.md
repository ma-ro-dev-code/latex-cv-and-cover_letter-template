# LaTeX CV and Cover Letter Template  

A clean, modern and fully customizable **LaTeX template for a CV and Cover Letter**.  
This repository provides two fully editable `.tex` files and corresponding compiled `.pdf` examples.  
All content is anonymized and can be easily adapted for any professional profile.
